## Run Locally

Download

- node
- npm
- typescript / tsc

Clone the project

```bash
git clone https://github.com/Kkwagh-Coders/interview-experience-ui.git
```

Change to Project Directory

```bash
  cd interview-experience-ui
```

Install Dependencies

```bash
  npm install
```

Start React App

```bash
  npm run start
```
